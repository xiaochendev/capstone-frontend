# Steps
1. Copy rerepository to your local file
```
git clone https://github.com/xiaochendev/gRepo.git
```

2. Install all the required dependencies
```
npm install
```

3. Start the server
```
npm start
```

4. Its viewable in your browser by entering
```
localhost:3000
```
